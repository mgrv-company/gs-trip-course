# 맹그로브 고성 · 로컬 트립 코스

투숙객이 QR코드로 접속해 **머무는 일수 / 이동수단 / 여행 목적 / 스타일 / 식성**을 고르면,
CA가 직접 다녀본 속초·고성 가게들로 맞춤 여행 코스를 만들어주는 모바일 웹페이지.

- 데이터: 개인 네이버지도 저장 리스트(술집/식사/카페) → 속초·고성 한정 181곳
- 기준점: 맹그로브 고성 (강원 고성군 토성면 교암길 20)
- 서버 없음 — 순수 정적 페이지 (GitHub Pages 무료 호스팅)

## 구조

| 파일 | 역할 |
|---|---|
| `index.html` | 설문 UI + 디자인 |
| `app.js` | 코스 생성 규칙 (식성·분위기·거리 점수 매칭) |
| `places.js` | 장소 데이터 (자동 생성) |
| `검수표.md` | 태그 검수용 표 (구역별 정렬) |
| `test/smoke.js` | Playwright 스모크 테스트 |

## 데이터 갱신 방법

네이버 저장 리스트에 가게를 추가한 뒤, Claude Code에서
"gs-trip-course 장소 데이터 갱신해줘"라고 요청하면
공유 링크에서 다시 긁어와 `places.js`를 재생성한다.

## 테스트

```
node test/smoke.js
```

(wings-phone-fix 프로젝트의 Playwright를 재사용)
